Basic

Lær det besale om HTML og CSS og lær hvordan man kombinere sprogne. HTML er byggestenene til din hjemmeside og css er det der bestemmer hvordan de skal se ud. Dette er et rigtig godt sted at starte hvis du er ny til kodning.

Start

Responsive og Sass

Lær hvordan du designer din hjemmeside så den er responsive, det vil sige sådan at den kan tilpasse sig forskellige enheder, uden at gå på kompromi med funktionaliteten. Derudover lærer du også sass, hvilket er en css preprocessor.

Javascript

Lær det besale i javascript, javascript kan øge funktionaliteten på din side og manipulere med elementer for at få siden til at blive mere dynamisk og flot. For at starte dette kursus er det en god ide med basis viden inden for html og css
